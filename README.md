## 注意点
- 一部を除き、無料版のDOTweenでは動作しません  
有料版のみの機能であるTextMeshProのアニメーション、一文字ごとのアニメーション(DOTweenTMPAnimator)を多く利用しています
- DOTweenPro、フォント、TextMeshProは含まれていないので各自で導入をお願いいたします
